# Freshly landing page

Responsive React landing page for Freshly, built with Vite, Motion, and Lucide React.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Source structure

```text
src/
├── assets/       Image registry and categorized media
├── components/   Reusable UI components
├── constants/    Shared animation configuration
├── data/         Product and location content
├── sections/     Landing-page sections
├── App.jsx       Page composition
├── main.jsx      React entry point
└── styles.css    Global and responsive styles
```
